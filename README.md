# Maya Black Card Rewards Converter

A lightweight, single-file web application designed to help users quickly convert between Philippine Pesos (PHP) and Maya Miles based on the standard redemption rate of **1 Maya Mile = ₱0.16**.

## Features

- **Bi-directional Conversion**: Convert Pesos to Miles or Miles to Pesos instantly.
- **Real-time Calculations**: Updates automatically as you type.
- **Dark-themed UI**: A modern, sleek interface inspired by the Maya Black Card aesthetic.
- **Zero-Dependency**: Built with pure HTML, CSS (Tailwind via CDN), and vanilla JavaScript.

## Getting Started

Since this is a single-file application, you do not need to install any development environments or run a server.

1.  **Download/Copy the code**: Save the provided code into a file named `index.html`.
2.  **Open in Browser**: Simply double-click `index.html` to open it in your preferred web browser (Chrome, Firefox, Safari, or Edge).

## How it Works

- **Pesos to Miles**: Enter an amount in PHP, and the app calculates how many Maya Miles are equivalent to that cash value based on the rate of ₱0.16 per mile.
- **Miles to Pesos**: Enter your current Miles balance, and the app shows you the equivalent cash value in PHP.

## Tech Stack

- **Frontend**: HTML5
- **Styling**: Tailwind CSS (loaded via CDN)
- **Logic**: Vanilla JavaScript (ES6+)
